# Minecraft Bedrock Addons

自用的 Minecraft 基岩版(Bedrock)行为包合集,纯脚本实现,不需要资源包。

| 行为包 | 一句话介绍 | 安装文件 | 最低游戏版本 |
|---|---|---|---|
| **Admin Wand(管理员权杖)** | 一根权杖弹菜单:满附魔、整理背包、昼夜切换、回家传送 | `AdminWand.mcpack` | 1.21.30 |
| **Buff Wand(祝福权杖)** | 常驻 Buff 权杖:勾选想要的 buff 自动续杯永不掉线,附带 255 级整活模式 | `BuffWand.mcpack` | 1.21.30 |
| **Max Enchant Helper** | 一条指令给装备附上全部满级附魔(权杖的前身,功能已并入权杖) | `MaxEnchantHelper.mcpack` | 1.21.30 |

## 📦 安装

1. 双击 `.mcpack` 文件,游戏会自动导入(或在游戏里 设置 → 存储 → 导入)。
2. 进入世界设置 → **行为包**,启用对应的包。
3. 世界需要**开启作弊**(权杖需要用 `/scriptevent` 指令获取)。

进入世界后,聊天栏会显示加载成功的横幅,看到横幅就说明包生效了。

## 🪄 Admin Wand(管理员权杖)

### 获取权杖

在聊天栏输入:

```
/scriptevent wand:give
```

会得到一根蓝色名字的「管理员权杖」(本体是烈焰棒)。**手持权杖**,手机长按 / PC 右键 / 手柄按使用键,菜单就会弹出。

> 注意:权杖是靠「改名烈焰棒」识别的,不要用铁砧改它的名字,改了就不认了。再要一根重新输指令即可。

### 菜单功能

| 菜单项 | 功能 |
|---|---|
| 一键满附魔 » | 二级菜单:**全部满附魔**(装备+背包里所有能附魔的物品)或**选择单件附魔**(从列表挑一件)。自动跳过并移除诅咒附魔。互斥附魔按固定优先级取舍:锋利(重锤为致密)、保护、时运、深海探索者、无限、多重射击、忠诚+引雷,物品上已有的低优先附魔会被替换 |
| 整理背包 » | 二级菜单,见下方详细说明 |
| 清理背包 | 清空背包(弹二次确认;权杖和穿着的盔甲会保留) |
| 切换白天/黑夜 | 在白天和黑夜之间一键切换 |
| 设置当前位置为家 | 记录脚下坐标+维度,退出重进不会丢 |
| 返回家 | 传送回已设置的家,支持跨维度 |

### 整理背包子菜单

| 选项 | 功能 |
|---|---|
| 一键整理 | 同类物品合并;工具武器排到快捷栏,方块、矿物、杂物依次靠后,食物固定在背包末尾;垃圾(腐肉、毒马铃薯)自动删除 |
| 矿物合成块 | 铁锭×9→铁块 等 12 种配方(铁/金/铜/钻石/绿宝石/红石/煤/青金石/下界合金/三种粗矿),零头保留 |
| 矿物块拆解 | 上面的反向操作;背包放不下的部分会掉在脚下并提示 |
| 存入附近箱子 | 把背包物品自动存进周围 6 格内的箱子/木桶,规则见下 |

### 箱子分类存储

在箱子**任意一面(含顶部)**紧贴放一块告示牌,写:

```
[storage]
ores
```

第一行固定 `[storage]`,第二行是分类标签,中英文都认:

| 标签 | 收纳内容 |
|---|---|
| `ores` / `矿物` | 矿石、锭、粗矿、宝石、矿物块 |
| `food` / `食物` | 所有食物 |
| `tools` / `工具` / `武器` | 工具和武器 |
| `blocks` / `方块` | 建筑方块 |
| `misc` / `杂物` | 不属于以上分类的东西 |
| `all` / `全部` | 什么都收 |

规则:

- **有标签的箱子**优先收货,装满自动溢出到下一个同类箱子。
- **没有标签的箱子**按「快速堆叠」逻辑:只收它**已经装有**的物品种类,不会乱塞。
- 权杖本体永远不会被存进箱子。

### 常见问题

- **菜单弹不出来?** 确认手持的是权杖本体(名字是蓝色的「管理员权杖」),且行为包已在世界设置中启用。
- **为什么拿着权杖也能附魔背包里的剑?** 附魔功能扫描的是整个背包+装备栏,不需要把物品拿在手上。
- **垃圾清单想加东西?** 编辑 `admin_wand_bp/scripts/main.js` 里的 `JUNK_IDS` 列表,重新打包即可。

## ✨ Buff Wand(祝福权杖)

### 获取权杖

在聊天栏输入:

```
/scriptevent buff:give
```

会得到一根粉色名字的「祝福权杖」(本体是微风棒)。**手持权杖**,长按 / 右键 / 使用键打开菜单。

> 和管理员权杖一样,权杖靠「改名微风棒」识别,不要用铁砧改名。

### 菜单功能

| 菜单项 | 功能 |
|---|---|
| 配置常驻 Buff » | 勾选式表单,勾上开启、取消勾选关闭,提交后立即生效 |
| 清除所有效果 | 清掉身上全部状态效果(包括中毒、凋零等负面效果),并同时关闭常驻配置 |

### 常驻模式说明

- 勾选的 buff 由脚本**每 20 秒自动续杯**,永不过期,药水粒子已隐藏。
- 配置存在玩家身上,**掉线重连、退出重进存档都会自动恢复**;多人各自独立。
- 夜视不会闪屏(时长始终保持在 40 秒以上)。

### 可选 Buff(默认等级)

速度 II、急迫 II、力量 II、抗性提升 II、生命恢复 II、跳跃提升 II、防火、水下呼吸、夜视、缓降、饱和、生命提升 II(+4❤)。

### 🎉 整活模式

勾选表单顶部的「整活模式」开关后,buff 按 **255 级**(游戏允许的最大等级)施加:力量一拳秒杀、急迫瞬间破坏、速度快到飞起……

- **跳跃提升是例外**:整活模式下按 VI 级施加,大约跳 6 格——255 级会跳上天要等半天才落地,实测不好玩。
- 防火、夜视等开关型效果没有等级,整活模式下不变。
- 玩腻了取消勾选即可恢复正常等级;「清除所有效果」也能一键清掉 255 级 buff。
- ⚠️ 跳 6 格落地也会掉一点血,怕摔可以搭配「缓降」或「生命恢复」。

## ⚔️ Max Enchant Helper

权杖出现前的独立附魔包,功能已并入权杖,单独用也可以:

```
/scriptevent max:enchant   # 给手持物品附满附魔
/scriptevent max:all       # 给全身装备附满附魔
/scriptevent max:debug     # 打印附魔诊断信息
```

互斥附魔的取舍规则与权杖一致(锋利、保护、时运、深海探索者、无限、多重射击、忠诚+引雷优先)。

## 🔧 从源码打包

行为包源码分别在 `admin_wand_bp/`、`buff_wand_bp/` 和 `max_enchant_bp/` 目录。修改后用 PowerShell 重新打包:

```powershell
Compress-Archive -Path .\admin_wand_bp\* -DestinationPath .\AdminWand.zip -Force
Move-Item .\AdminWand.zip .\AdminWand.mcpack -Force
```
